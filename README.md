# MacroPredict Market

A data-intensive financial prediction market platform focused on macroeconomic events.

## Project Structure

- `backend/`: FastAPI backend implementation.
- `frontend/`: React + Tailwind frontend application.
- `docs/`: Product and technical documentation.
- `data/`: Data ingestion and analysis pipelines.
- `infra/`: Infrastructure as Code and deployment configurations.
- `antigravity/`: AI-assisted coding prompts and instructions.

## Getting Started

### Prerequisites
- Python 3.10+
- Node.js 18+
- Docker & Docker Compose

### Development
1. Clone the repository.
2. Copy `.env.example` to `.env`.
3. Follow instructions in `backend/README.md` and `frontend/README.md`.
